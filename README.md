# Zoodini

## Description
Help Gar the Cat and Otto the Octopus use stealth and strategy to outmaneuver guards and free their animal friends from captivity!

Under the cover of night, Gar, a fearless cat, and Otto, a cunning octopus stolen from the ocean, orchestrate a daring zoo breakout. Help our fearless furry friends sneak past guards, disable security cameras, unlock gates, and hide under manhole covers to swiftly move through the high-security zoo and rescue their friends from captivity.

## Installation

Download the latest build from the [Releases page](../../releases/latest).

### Windows
1. Download **`Zoodini-win.zip`** from the latest release.
2. Right-click the downloaded file and select **Extract All...** (or use your preferred archive tool).
3. Open the extracted folder and double-click the **Zoodini** executable to play.

> **Note:** Windows Defender SmartScreen may warn you that the app is from an unknown publisher. Click **More info → Run anyway** to continue.

### macOS
1. Download **`ZoodiniPackageMac-*.zip`** from the latest release.
2. Double-click the downloaded file to unzip it.
3. Move **Zoodini.app** to your **Applications** folder (optional, but recommended).
4. The first time you open the game, macOS Gatekeeper may block it since the app isn't notarized. To bypass this:
   - **Right-click** (or Control-click) the app and select **Open**, then click **Open** in the dialog that appears.
   - Or, run the following in Terminal:
     ```bash
     xattr -cr /path/to/Zoodini.app
     ```
5. Launch the game and enjoy!

### Building from Source
1. Download the **Source code** archive from the release (or clone the repository):
   ```bash
   git clone https://github.com/SebastianJeanF/Zoodini.git
   ```
2. Open the project in your IDE and build/run from there.
